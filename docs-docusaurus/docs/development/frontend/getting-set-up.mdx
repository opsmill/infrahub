# Getting set up with frontend

:::warning Before we start

Make sure [Infrahub Backend](../backend.mdx) is up and running. If not, in your terminal execute:

```sh
invoke demo.destroy demo.build demo.start demo.load-infra-schema demo.load-infra-data
```

:::

Infrahub is built with React. Make sure you're running Node.js 20+ and NPM 9+, to verify do:

```sh
node --version
npm --version
```

## 1. Install dependencies

```sh
cd /frontend
npm install
```

## 2. Start a local server

```sh
npm start
```

To can access your local server at [http://localhost:8080/](http://localhost:8080/). If you are not familiar with Infrahub, learn by following our [tutorial](/tutorials/getting-started/).

## 3. Run all tests

### Unit tests

```sh
npm run test

# same with coverage
npm run test:coverage
```

### Integration tests

```sh
npm run cypress:run:component
```

### E2E tests

```sh
npm run test:e2e
```

All tests should succeed. For more information on testing, read [Running & Writing Tests](testing-guidelines.mdx).
