# Welcome to Infrahub

Infrahub from [OpsMill](https://opsmill.com) is taking a new approach to Infrastructure Management by providing a new generation of datastore to organize and control all the data that defines how an infrastructure should run. Infrahub offers a central hub to manage the data, templates and playbooks that power your infrastructure by combining the version control and branch management capabilities of Git with the flexible data model and UI of a graph database.

## Learn about Infrahub

- Infrahub [Tutorials](tutorials/) explain how the platform is designed and walk you through learning it's basic features with a demo dataset.
- Infrahub [Guides](guides/) comprise a comprehensive set of "how-to" topics, covering the key areas of Infrahub with which users will engage.
- Infrahub [Topics](topics/) provide a more in-depth technical explanation of various features, implementation, and deployment details.
- Infrahub [Reference](reference/) documents technical implementation details such as available options for features and commands.
